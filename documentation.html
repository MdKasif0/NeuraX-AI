Getting Started

Welcome to the Chatbot NeuraX AI documentation. This guide will help you get started with integrating our powerful AI-driven chatbots into your applications.

Quick Start


// Initialize the Chatbot NeuraX AI
const chatbot = new ChatbotBuilderAI({
    apiKey: 'YOUR_API_KEY',
    botId: 'YOUR_BOT_ID'
});

// Start a conversation
chatbot.startConversation();

API Reference

Our RESTful API allows you to interact with the Chatbot NeuraX AI platform programmatically. Here are some key endpoints:

  • GET /api/v1/bots - List all your chatbots
  • POST /api/v1/conversations - Start a new conversation
  • PUT /api/v1/messages - Send a message in a conversation

Webhooks

Set up webhooks to receive real-time notifications about events in your chatbot:


POST /api/v1/webhooks
{
    "url": "https://your-app.com/webhook",
    "events": ["message.created", "conversation.ended"]
}
